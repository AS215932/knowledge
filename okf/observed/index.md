# Observed evidence

Manual/local read-only observations are written here. `uv run hyrule-knowledge observe --profile safe-health` can write the default `safe-health.md` snapshot when local environment variables are configured.

Observed concepts are evidence-only, expire quickly, and must not be treated as canonical intended state.

* [Latest observations](latest/) - Most recent sanitized snapshots.
* [Update log](log.md) - Human-readable observation ingestion log.
